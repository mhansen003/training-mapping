# CLEAR 2.0 - System Documentation

> CMG Financial's Core Loan Origination System

---

## Overview

CLEAR 2.0 is the primary loan origination system used by CMG Financial for processing mortgage applications. This documentation covers all major modules and workflows.

## Primary Users

| Role | Primary Modules |
|------|-----------------|
| **Loan Officer** | Scenario Pricing, Pipeline Views, Borrower Management, Credit Ordering |
| **Processor** | Borrower Needs List, Condo Processing, Disclosure History |
| **Operations** | AUS Services, Pipeline Management |
| **Sales** | Sales Dashboard, Opportunity Loans |
