AUS Services in CLEAR 2.0

Automated Underwriting System (AUS) services in CLEAR 2.0 allow Loan Officers to submit loans to DU (Desktop Underwriter) for Fannie Mae or LPA (Loan Product Advisor) for Freddie Mac to receive automated underwriting decisions. DU Early Access (DU EA) is available for prequalification scenarios.

Desktop Underwriter (DU)

DU is Fannie Mae's automated underwriting system for conventional loans.

Loan TypeCredit RequiredAUS Type
ConventionalSoft Pull or Tri-MergeDU / LPA
FHATri-Merge RequiredDU Only
VATri-Merge RequiredLPA Only

  1. Navigate to AUS

    Click on the DU/LPA tab in CLEAR or access via the Action Bolt dropdown.

  2. Select DU

    Choose Desktop Underwriter from the AUS options.

  3. Run AUS

    Click "Run AUS" to submit. Results typically return within seconds.

  4. Review Findings

    Review the DU Findings report for approval status and conditions.

Loan Product Advisor (LPA)

LPA is Freddie Mac's automated underwriting system. Use LPA for conventional loans targeting Freddie Mac delivery or for specific investor requirements.

VA loans must use LPA as DU is not available for VA loan products.

DU Early Access (DU EA)

DU EA allows running Desktop Underwriter during prequalification with a soft pull credit report and TBD property address. This helps determine borrower eligibility before a property is identified.

üí° When to Use DU EA
  • Prequalification scenarios without a property
  • Early assessment of borrower qualification
  • Soft pull credit only available
