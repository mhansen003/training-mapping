What is the Processor Certification?

The Processor Certification is an internal use only document used by CMG Financial to certify that a loan processor has completed required tasks and verification steps on a specific loan file. This form serves as an internal quality control checkpoint.

When to Complete the Processor Certification

  • Pre-Submission: Complete the certification after all processor tasks are finalized but before submission to underwriting
  • Quality Control Check: Use as part of the internal QC process to verify all required documentation is complete
  • File Audit Trail: Provides documentation of processor accountability for the loan file
  • Compliance Verification: Confirms adherence to CMG Financial processing standards

Internal Use Only

This document is marked "INTERNAL USE ONLY" and should not be shared with borrowers, title companies, or other external parties. It remains part of the internal loan file documentation.

Data Accuracy

Ensure all information matches exactly what is recorded in the Loan Origination System (LOS). The loan number and borrower name should be copied directly from the loan file to prevent discrepancies.

  • Step 1: Access the Processor Certification form template from the CMG Financial document repository or LOS
  • Step 2: Enter the current Date in MM/DD/YYYY format
  • Step 3: Enter the Loan Number exactly as it appears in the LOS
  • Step 4: Enter the Borrower Name (primary borrower's full legal name)

  • Step 6: Save or print the completed form for the loan file

Processor Signature

The "Thank you" line at the bottom of the form indicates acknowledgment by the processor. Some workflows may require an additional electronic or physical signature depending on branch requirements.

The Processor Certification becomes part of the permanent loan file and audit trail. Ensure the document is properly indexed and stored so it can be retrieved during quality control reviews or audits.

Retention Requirements
