Understanding Pre-Qual and Mortgage-Only report requirements

When is This Required?

Certain loan programs allow a soft-pull credit check for initial underwriting but require a Mortgage-Only report before proceeding to Clear-to-Close. The Mortgage-Only report can be converted from a Pre-Qual report on the Xactus website.

Credit Report Types Explained

  • Pre-Qualification Report (Soft Pull): Used for initial loan qualification. Does not impact borrower's credit score. Suitable for early-stage underwriting assessment.
  • Mortgage-Only Report (Hard Pull): Required for final underwriting and Clear-to-Close. This is a formal credit inquiry that appears on the borrower's credit history.
  • Conversion Process: Transforms the existing Pre-Qual report into a Mortgage-Only report while retaining borrower data. FICO scores are included in the conversion by default.

Key Benefit

Converting from Pre-Qual to Mortgage-Only allows you to use the same borrower information without re-entering data, saving time and reducing errors in the credit ordering process.

Actions Menu Options

The Actions menu on the left side includes various options for managing the credit order:

  • Copy/Merge/Unmerge: Combine or separate credit reports
  • Add Repositories: Add additional credit bureaus
  • Order Supplement/X: Request supplemental credit information
  • Compare Reports: Compare multiple credit reports
  • Submit Service Request: Contact support
  • Download Loan Data/X003: Export loan data
  • Reassign Report Owner: Transfer ownership
  • Cancel Report (Allow Repull): Cancel and enable new pull
  • Upload Supporting Docs: Attach documents
  • Convert to Mortgage Only: THE KEY ACTION for this process

Automatic Processing

Once you click "Convert to Mortgage Only," the system automatically processes the conversion. FICO scores are included in the conversion by default. You do not need to take any additional action for the basic conversion.

The FICO scores can be excluded once the report is converted. To exclude scores:

# Pre-Qual to Mortgage Only Report - Xactus 360

## Overview
Certain loan programs allow a soft-pull credit check for initial underwriting but require a Mortgage-Only report before proceeding to Clear-to-Close. The Mortgage-Only report can be converted from a Pre-Qual report on the Xactus website.

## Step-by-Step Process

### 1. Access Xactus 360
- Log into Xactus 360 website with CMG credentials
- Navigate to credit order dashboard

### 2. Locate Credit Order
- Search by loan number, borrower name, or SSN
- Open the credit order
- Verify report type is "Pre-Qualification X"

### 3. Convert to Mortgage Only
- On the left-hand side, locate the Actions menu
- Click "Convert to Mortgage Only"
- Report will automatically open when completed
- FICO scores are included in the conversion

### 4. FICO Score Options (Optional)
- Scores are included by default
- To exclude: Click (exclude) next to "Scores Included"
- Once excluded, scores are removed from the report

### 5. Review and Download
- Verify borrower information is correct
- Download the converted PDF report
- Upload to loan file as required

## Key Information
- Report Type changes from Pre-Qualification to Mortgage-Only
- All three bureau scores (TransUnion, Experian, Equifax) included
- Original Pre-Qual report remains available for reference
- "Convert Back to Pre-Qualification/X" option available if needed
