Pipeline File Designations

Each loan in the pipeline is assigned a status code that indicates its current stage in the processing workflow. Understanding these codes is essential for prioritizing work.

Code Status Name Description
RNW Ready - New New file ready for initial processing setup
RWE Ready - With Exceptions File ready but has outstanding exceptions to address
RCE Ready - Clear to Close All conditions cleared, ready for closing
REA Ready - Escalated File requires escalation or supervisor review
RNE Ready - No Exceptions File in good standing with no outstanding items
RSR Ready - Suspended/Resubmit File on hold pending resubmission or additional docs

Quick Info & Sandbox Mode

Quick Info: Access loan summary data without opening the full file. Sandbox Mode: Practice environment for training - changes made in Sandbox do not affect live loan data.

Certificate of Eligibility (COE)

The COE verifies the veteran's eligibility for VA loan benefits and shows available entitlement. Many COEs can be obtained automatically through WebLGY; others require manual request from the borrower.

  • Step 1: Log into WebLGY portal with CMG credentials
  • Step 2: Navigate to "Eligibility" section
  • Step 3: Enter veteran's SSN and consent information
  • Step 4: If COE available, download and store in Byte
  • Step 5: For VA Case #, navigate to "Case Number Assignment"
  • Step 6: Enter property and loan details
  • Step 7: Submit and document case number in loan file

VA Appraisal Ordering

VA appraisals must be ordered through WebLGY. The appraisal is assigned to a VA-approved appraiser and cannot be transferred to another property.

What is CAIVRS?

CAIVRS (Credit Alert Interactive Voice Response System) checks for delinquent federal debt, including defaulted government loans. Required for all FHA and VA loans before closing.

Result Meaning Action Required
Clear No delinquent federal debt found Document result and proceed
Hit Delinquent federal debt identified Escalate to Senior Processor immediately

CAIVRS Hit Resolution

A CAIVRS hit does not automatically disqualify the borrower. The debt may have been paid, forgiven, or incorrectly reported. Escalate to processor for research and resolution.

LoanSafe Fraud Manager

LoanSafe analyzes loan data for fraud indicators including identity fraud, income misrepresentation, property issues, and occupancy fraud. Run at initial processing and before closing.

  • Step 1: Navigate to LoanSafe from Byte menu
  • Step 2: Select loan and click "Run Fraud Check"
  • Step 3: Review all alerts and risk scores
  • Step 4: Document findings in Processing Log
  • Step 5: Escalate any high-risk alerts to supervisor

LoanSafe Resubmission

If loan data changes significantly (borrower added, property change, income update), resubmit LoanSafe to ensure accurate fraud analysis with current information.

The Work Number (TWN)

The Work Number is an automated employment verification service. Many large employers participate, allowing instant verification of employment dates and income without contacting the employer directly.

  • Step 1: Access Credit Plus through Byte integration
  • Step 2: Select "The Work Number VOE" service
  • Step 3: Enter borrower SSN and employer information
  • Step 4: Submit request and receive instant report (if employer participates)
  • Step 5: If employer not found, proceed with traditional verification methods

Re-Verifying Employment

Employment must be re-verified within 10 days of closing. Use TWN or VVOE to confirm borrower is still employed. Document re-verification in loan file.

Sentilink SSA-89

Sentilink verifies that the borrower's SSN matches Social Security Administration records. This helps prevent identity fraud and ensures loan eligibility for government programs.

Service Purpose When to Order
Sentilink SSA-89 Verify SSN matches borrower identity Initial processing for all loans
Payoff Request Get current payoff amount for existing liens When refinancing or paying off liens at closing

Ordering Payoffs

Request payoffs early in processing. Payoff statements are typically valid for 10-30 days. Include per diem interest and any prepayment penalties in calculations.

Equifax Tax Transcripts

Tax transcripts (4506-C) verify income reported on tax returns matches IRS records. Required for most loan types to validate borrower income documentation.

  • Transcript Ordering: Access Equifax through Byte, enter borrower info and tax years needed, submit 4506-C request
  • UDM Setup: Undisclosed Debt Monitoring automatically monitors for new credit inquiries and accounts after initial credit pull
  • UDM Alerts: Review any alerts for new debt and escalate to processor for re-qualification if needed

UDM Alert Resolution

New debt discovered through UDM must be disclosed and may require re-qualification. Common causes include new car loans, credit cards, or co-signed debts. Always escalate UDM alerts promptly.
