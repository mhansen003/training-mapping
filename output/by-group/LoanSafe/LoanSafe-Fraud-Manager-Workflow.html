SSA-89 Requirements

  • Both pages of SSA-89 form required
  • Signed and dated within 90 days
  • Name must match Byte EXACTLY
  • Electronic signatures acceptable via SentiLink

If Validation Fails

Confirm borrower's name, DOB, and SSN match their records with Social Security Administration. If still failing after confirmation, email LoanInvestigations@cmgfi.com.

Quick Reference Card

ACCESS: Interfaces > Fraud Detection > CoreLogic Fraud

WHEN TO RUN:
1. Initial: Auto at File Received status
2. Re-Run: When new parties added
3. Final: At CTC Requested (all parties confirmed)

RESPONSIBILITIES:
- Processor: Clear LOW alerts, add parties, leave notes
- Underwriter: Clear MEDIUM/HIGH alerts, fraud score >900

CLEARING ALERTS:
- Explained: Documentation obtained/in file
- Data Corrected: Fixed incorrect Byte data
- False Positive: Compliance Team ONLY
- Approved Exception: Compliance Team ONLY

SSN VERIFICATION:
- Both pages SSA-89 required
- Signed within 90 days
- Name must match Byte exactly
- Order via Interfaces > Verifications

EXTERNAL SEARCHES:
- ASC: https://www.asc.gov/appraiser/advanced#
- OFAC: https://ofac.treasury.gov/sanctions-list-search
- SAM: https://www.sam.gov (Exclusions)
- HUD LDP: hud.gov/program_offices/general_counsel/LDPList
- FHFA SCP: fhfa.gov Suspended Counterparty Program
