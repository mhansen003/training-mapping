üîê Access Truv in CLEAR

1

Open the Loan File

Access the loan in CLEAR 2.0 by navigating to the loan in your pipeline or using Global Search.

2

Navigate to VOE

Option A: Click the VOE button in the URLA Employment Section.
Option B: Go to Hamburger Menu ‚Üí Verifications ‚Üí VOE

‚ö†Ô∏è Important

Orders placed via SSO are no longer automatically imported into CLEAR. Always place Truv orders directly within CLEAR to ensure full automation including document retrieval and AUS reference number assignment.

üë§ Borrower Information

1

Select Borrower

Select the appropriate borrower from the dropdown. Information will auto-populate from the URLA.

2

Verify Contact Info

Confirm First Name, Last Name, Email, and SSN. Updates made here will save back to the URLA.

3

Add SMS (Optional)

Toggle Add SMS to send text message invitations. Orders with Email + Mobile complete 2x faster.

üí° Pro Tip

If your borrower does not want text messages, leave Add SMS toggled off. They will receive email-only invitations.

üîç Coverage Check

1

Search for Employer

Click "Click to view Truv's Coverage" and search for the borrower's employer name.

2

Review Coverage Indicator

Before selecting, check the color-coded coverage indicator to assess connection likelihood.

High Expected to succeed - send the order
Low May succeed if borrower knows their payroll provider
Not Verified Contact borrower to identify payroll provider
Not Supported Use alternative verification method

üì§ Submit Order

1

Select Employer(s)

Find and select the borrower's employer and/or payroll provider. You can add multiple employers if needed.

2

Click Submit

Click the Submit button to process the order. Borrower will receive email (and SMS if enabled) invitations.

üìß Borrower Notifications

Once submitted, the borrower receives an email with "Verify Instantly" button. If SMS enabled, they'll also get a text with direct link. Completion emails are sent to order creator and LO when results are ready.

‚öôÔ∏è Order Management

1

2

Available Actions

Email Icon: Resend invitation link
View Timeline: See detailed order status in Truv Dashboard
Retrieve: Refresh CLEAR if connection made instantly
Import: Add or Replace URLA employment info

‚ö†Ô∏è Income Entry Required

Income will NOT be imported automatically. Review the VOIE Report in Clear Docs and enter income manually in the URLA employment section.
