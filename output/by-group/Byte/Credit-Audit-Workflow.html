Buydown Configuration

How to identify a Buydown?
CD Page 2 will show a buydown, sometimes the purchase agreement

FEMA Disaster Area Check

If FEMA Disaster Area is checked but it's been 120 days since disaster, confirm the address is still in a disaster area on FEMA website and internal list. If no longer in disaster area, uncheck the box.

USDA Annual Fee

The USDA Annual fee will no longer need to be entered as a new fee. This fee will be shown under the Mortgage Insurance Fee on the Waiver of Escrow screen. Months need to show 0, Disburse Annually, and the starting month/year on the IEAD.

Conditions Added

Important Condition Rules

  • No Duplicates: Cannot use duplicate conditions - add all required items for same document type in one condition
  • Deleting: Right-click condition and click delete - only possible BEFORE loan approval
  • After Approval: Condition must be cleared with commentary explaining why waived
  • Alphabetical Order: Use Ctrl+F with condition number to locate

2nd Level / Exception Request Process

When is 2nd Level Review Required?
Search for "Pending 2nd" in Add Conditions from Templates
Choose "Pending 2nd level review due to:" and complete reason
Example: First time homebuyer using rental income to qualify
Requesting Exception
Double-click condition, click Exception Request button (adds your name)
Write-up goes into Conversation Log
Management updates Exception Status and satisfies condition

Trailing Conditions (Post-Purchase)

To move a condition to Post-Purchase/Trailing Docs:

  • Double-click on the condition
  • Change Class to: TRAIL
  • Change Stage to: Trailing Docs
  • Add a 30 day Follow Up Date

Conditions for Review Date

When Conditional Review is complete, remove the "Conditions for Review Date". When CORR - Audit Complete - Conditions Pending is checked, CLEAR is updated to notify sellers of updated approval.

Lock Desk Update Requirements

Include loan number in entirety and borrower name (e.g., CSR0000066, Wilson). These fields should be locked once loan is locked:

  • Base Loan Amt and Final Ln Amts
  • Rate
  • Loan Program
  • Loan Term
  • FICO - If Median Credit Score doesn't match Estimated Credit Score, email lock desk
  • Notes dated over 45 days from Note date
  • Manual Underwrites

Credit Audit Screen - Clearing Conditions

  • Remove Credit Condition Uploaded Date when conditional review complete (removes from pipeline)
  • Check "All Credit Conditions Have Been Cleared" when all cleared

Vacancy Factor Override

When to Override?
When updating REO section to match AUS
Double-click on 25% to edit
Tax Returns → Override to 0
Lease Agreements → Keep 25%

Excluding Debts from Qualifying

To exclude any debt from qualifying, check BOTH:

  • "Not Counted"
  • "Omitted"
  • If being paid off, also check "To Be Paid Off"

Other Borrower Responsible for Mortgage

Multiple Borrowers - Same Mortgage

When mortgage shows for both borrowers and should only count once:

  • Go to Liabilities > Show All Liabilities
  • Double-click mortgage showing for more than one borrower
  • Change Account Type to Installment
  • Enter $0.00 for Mo. Payment

Non-Rental Property Handling

StepActionResult
1Change Status from "Retained Non-Rental" to "Retained - Rental"Enables income/vacancy fields
2Override Vacancy Factor to 0Removes vacancy calculation
3Override Net Rental Income to 0Payment no longer included in qualifying

URLA Lender Loan Info - Subordinate Financing

Subordinate Financing Location

Subordinate Financing is listed in URLA Lender Loan Information under Proposed PITI, 2nd line item. Click Edit to see Subordinate Lien information.

Proposed PITI - "Other Amount" Fix

Other Amount Must Be Categorized

If payment showing in "Other" field:

  • Click Edit for Other
  • Review file to identify what amount is for
  • Click dropdown for Options (do NOT choose Other)
  • Select appropriate category (e.g., Flood Insurance)
  • Nothing can be left in "Other"
