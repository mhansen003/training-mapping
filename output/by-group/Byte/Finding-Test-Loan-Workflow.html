Security Profile Selection

Which Profile to Choose?
Corporate Account - Access to ALL test loans across organization
Training Branch - Access to training-specific test loans
Specific Branch - NO access to test loans (production only)

Important: Branch Selection

Choosing a specific branch will NOT allow access to test loans. You must select a Corporate or Training account for training/testing purposes.

Pro Tip: Switching Profiles

Your Security Profile can be switched after logging in. Navigate to File > Change Profile and select the desired profile from the list.

# Finding a Test Loan in BytePro Enterprise

## Overview
All loans are housed in specific pipelines and are not visible unless the correct profile and pipeline are selected. Follow the below instructions to access your test loan.

## Step 1: Login to BytePro Enterprise
- Open BytePro Enterprise application
- Enter your credentials:
  - **User Name**: lastnamefirstname (e.g., francescaleehive)
  - **Password**: cmg1234 (initial password - change on first login)
  - **Connection**: Production
- Click OK to login

## Step 2: Select Security Profile
- When prompted, select a Security Profile
- **For training/testing purposes, select a Corporate or Training account**
- **IMPORTANT**: Choosing a specific branch will NOT allow access to test loans
- To change profile after login: File > Change Profile

## Step 3: Navigate to Test Loans Pipeline
- Test loans can be found in various pipelines depending on your channel and role
- Choose the **Test Loans Pipeline** from the blue pipeline tiles
- Alternative: Test loans can also be found in **All Loans**

## Step 4: Search and Access the Loan
1. Change the **Matching** dropdown from "Exact" to "Within"
   - *This only needs to be done one time - Byte will save this setting*
2. Type or paste the loan number in the **File Name** field
3. Click **Search** or press **Enter** on the keyboard
4. The loan displays in the row below
5. **Double-click** the loan row to access the loan

## Quick Reference
- Username format: lastnamefirstname
- Initial password: cmg1234
- Profile needed: Corporate or Training (NOT specific branch)
- Pipeline: Test Loans (blue tile) or All Loans
- Search setting: Change Matching to "Within" (one-time)
- Access loan: Double-click the row
