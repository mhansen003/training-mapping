Disaster Protection Overview

Understanding the multi-layer protection system

Purpose

CMG has implemented additional checks throughout the loan process to identify and review properties in a Disaster Area. Through email notifications, conditions, and system checks, the goal is to eliminate unknown disaster area loans moving forward without missing additional documentation.

Disaster Attestation

Appraisal Info Screen disaster area fields

Location in Byte

A Disaster Area section has been added to the header of the Appraisal Info Screen within Byte.

Disaster Area Fields

Field Type Description
Disaster Incident Begin Date Read Only Start date of the disaster incident
Disaster Incident End Date Read Only End date of the disaster incident (if applicable)
Disaster Declaration Date Read Only Official FEMA declaration date
Declaration Title Read Only Name of the disaster (e.g., WILDFIRES)
State Read Only State of the disaster area
Designated Area Read Only County or specific designated area
Acknowledgment Checkbox Required "I acknowledge I have received and reviewed the Disaster Report/Requirements"

Processor Responsibility

Processors are responsible to complete the attestation in the Appraisal Info Screen by checking the acknowledgment checkbox after reviewing the Disaster Report/Requirements.

PROP 609 Condition

Automated underwriting condition for disaster properties

PROP 609 Trigger Logic

Condition Trigger
IF Disaster Incident Begin Date is AFTER Appraisal Completed Date
AND Loan is NOT yet CTC
THEN Condition PROP 609 is automatically inserted

Hard Stop Process

Funds Ordered attestation requirement

Critical Hard Stop

If the disaster attestation checkbox is NOT checked, the loan cannot move to the next status if in Funds Ordered. A hard stop will prevent progression.

Hard Stop Flow

Loan at Funds Ordered Status
|
Attestation Checkbox Checked?
YES |
Proceed to Next Status
NO |
HARD STOP

Hard Stop Error Message

"This file has been flagged to be in a current emergency disaster area. The "I acknowledge I have received and reviewed the Disaster Report/Requirements" checkbox is a required field. Please review/complete to continue"

Resolution Steps

To resolve the hard stop:

  1. Navigate to the Appraisal Info Screen
  2. Review the Disaster Report/Requirements
  3. Check the acknowledgment checkbox
  4. Return to move the loan to the next status
