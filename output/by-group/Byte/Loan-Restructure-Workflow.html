When to Use Loan Restructure

Criteria for requesting status change

üìã General Information

Loan Restructure is a Byte-specific status. When a loan is in Loan Restructure status, loan parameters cannot be edited in Clear.

Processors are responsible for updating the status to Loan Restructure upon your request.

Use Loan Restructure For:

üí∞ Income/Employment Changes

Changes requiring recalculation of qualifying income

üè¶ Asset/Funds Updates

Funds-to-close updates affecting qualification

üìä Loan Amount/LTV Changes

If loan is NOT locked (see locked instructions below)

üîÑ AUS Re-run Required

Any changes that require AUS to be re-run

‚ö†Ô∏è Important: When the Loan is LOCKED

Any field that could affect locked terms must be completed in Optimal Blue:

  • Interest rate or price changes
  • Loan Amount or LTV changes
  • Lock term/expiration changes
  • Lock extensions, relocks, or renegotiations

  • Pricing updated (if locked)

  • Married borrowers = same application
  • Tab out to auto-save
