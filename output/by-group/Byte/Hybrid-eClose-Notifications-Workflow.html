What is Hybrid e-Close?

Hybrid e-Close is a time-saving closing option that allows borrowers to electronically sign certain documents before the closing appointment. A series of automated emails are generated by Byte to indicate loan eligibility and status changes throughout the process.

Hybrid e-Close is NOT allowed with Power of Attorney OR Trust. If either is added to the loan, the system automatically sends a "No Longer Eligible" notification.

# Hybrid e-Close Notifications

## Overview
A series of emails are auto-generated by Byte to indicate loans eligible for Hybrid e-Close.

## Eligibility Triggers
The system evaluates eligibility when the loan reaches:
- UW Received
- Conditions for Review
- Changes to UW

## Eligibility Requirements
- Eligible loan program
- Standard vesting type
- **NOT allowed:** Power of Attorney
- **NOT allowed:** Trust

## Email Types

### 1. Eligibility Notification Email
**Recipients:** LO, Processor, Production Assistant, JR LO
**Subject:** "[Loan #], [Name], Your loan is eligible for Hybrid E-Close."
**Sent when:** Program and vesting type are eligible

### 2. No Longer Eligible Email
**Recipients:** LO, Processor, Production Assistant, JR LO
**Subject:** "[Loan #], [Name], Your loan is no longer eligible for Hybrid E-Close."
**Sent when:** POA added, Trust added, or loan program changed

### 3. Hybrid e-Close Requested Email
**Recipients:** Closer, Funder
**Subject:** "[Loan #], [Name], Hybrid E-Close has been requested on your loan"
**Sent when:** User enters Hybrid E-Close Requested date in Byte

### 4. Hybrid e-Close Cancelled Email
**Recipients:** Closer, Funder, LO, Processor, Production Assistant, JR LO
**Subject:** "[Loan #], [Name], Hybrid E-Close has been cancelled on your loan"
**Sent when:** User enters Hybrid E-Close Cancelled date in Byte

## Screen Locations
- **Retail CD Request** - All e-Close fields viewable
- **Document Preparation (Header)** - Enter dates
- **Funding Review (Header)** - Above Important Dates
