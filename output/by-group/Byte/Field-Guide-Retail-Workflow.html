Quick Reference Card

BYTE FIELD GUIDE - QUICK LOOKUP
================================

BORROWER DATA:
  Personal Info        → URLA Borrower Info > Personal Information
  Income               → URLA Borrower Info > Employment and Income
  Assets               → URLA Borrower Info > Assets
  Liabilities          → URLA Borrower Info > Liabilities
  Declarations         → URLA Borrower Info > Declarations

LOAN DATA:
  Loan Amount/Rate     → Overview Panel (quick) or URLA Lender Loan Info
  LTV/DTI Ratios       → Overview Panel or 1008 Transmittal
  Case Numbers         → Home Screen or URLA screens > Loan Identifiers
  Rate Lock            → Home Screen > Registration and Lock Info

PROPERTY/APPRAISAL:
  Property Type        → Overview Panel or 1008 Transmittal
  Appraisal Dates      → Appraisal Info > Appraisal Info 1
  VA Appraisals        → Appraisal Info > VA Appraisals

DISCLOSURES:
  LE History           → Loan Estimate Page 1 > Disclosure History
  CoC Requests         → Doc Prep or Changed Circumstance screen

CLOSING:
  Title Vesting        → Closing > General or Retail CD Request
  Seller Info          → Closing > Sellers or Doc Prep > Builder/Seller
  Wire Amount          → Funding > Funding Wire Calculations

BYTE FIELD GUIDE (RETAIL) - COMPLETE REFERENCE
===============================================

This guide maps data fields to their locations in the Byte LOS system.
Updated: October 28, 2022

BORROWER INFORMATION
--------------------
AKAs / Alternative Names
  - Closing > AKAs > AKAs - Credit Aliases
  - URLA Borrower Info > Personal Information > Alternative Names

Borrower Contact Information
  - URLA Borrower Info > Personal Information

Income
  - URLA Borrower Info > Employment and Income

Assets
  - URLA Borrower Info > Assets

Earnest Money Deposit
  - URLA Borrower Info > Assets > Earnest $
  - Application Summary > Details of Transaction > Earnest Money
  - Loan Estimate Page 2 > Details of Transaction > Earnest Money

Liabilities
  - URLA Borrower Info > Liabilities

Declarations
  - URLA Borrower Info > Declarations

LOAN DETAILS
------------
Loan Amount
  - Overview Panel > Loan
  - URLA Borrower Info > Loan and Property Information > Base Loan
  - URLA Lender Loan Info > Details of Transaction

Interest Rate
  - Overview Panel > Loan
  - Loan Estimate Page 1 > Loan Type > Interest Rate
  - URLA Lender Loan Info > Mortgage Loan Information > Int Rate
  - Application Summary > Loan Info > Note Rate

LTV
  - Overview Panel > Loan > LTV
  - Application Summary > Loan Info > LTV

DTI Ratios
  - 1008 Transmittal > III. Underwriting Information > Qualifying Ratios
  - Overview Panel > DTI Ratios
  - Application Summary > Loan Info

FHA/VA Case Number
  - Home Screen > Agency Case No:
  - URLA Borrower Info > Loan Identifiers > Agency Case No:
  - URLA Lender Loan Info > Loan Identifiers > Agency Case No:

Rate Lock
  - Home Screen > Registration and Lock Info > Lock Exp. Date
  - Overview Panel > Lock Expiration
  - Status Screen/Panel > Rate Expiration

PROPERTY & APPRAISAL
--------------------
Property Type
  - Overview Panel > Loan > Prop Type
  - Home Screen > Subject Property Info > Property Type
  - 1008 Transmittal > I. Borrower and Property Information

Appraised Value
  - 1008 Transmittal > I. Borrower and Property Information > Appr Value

Appraisal Ordered Date
  - Appraisal Info > Appraisal Info 1 > Appraisal Ordered
  - Processing Log > Appraisal Ordered

Appraisal Received Date
  - Appraisal Info > Appraisal Info 1 > Appraisal Received

VA Appraisal Dates
  - Appraisal Info > VA Appraisals > VA 1004D Requested
  - Appraisal Info > VA Appraisals > VA 1004D Received

TITLE & CLOSING

  - Retail CD Request > Vesting Description

  - Underwriting > Vesting
  - URLA Lender Loan Info > Title Information

Wire Amount
  - Funding > Funding Wire Calculations > Actual Wire Amount

First Payment Date
  - Loan Estimate Page 2 > Critical Dates
  - Closing Disc Page 2 > Critical Dates
  - Doc Prep > First Payment Date
