Critical: E-Signed Documents Warning

Byte REJECTS outside vendor electronically signed documents (DocuSign, DotLoop, etc.) in bundles when dragged and dropped. These include:

  • Purchase Agreements & Addendums
  • Letters of Explanation (LOEs)
  • College Transcripts

Solution: Use BytePro Print to Loan File for ALL electronically signed documents. Do NOT drag and drop.

When to Merge

Use merge when incomplete files are uploaded at separate times and need to be combined into one complete document. After merging, deactivate the old documents to avoid confusion during review.

Merge Process Summary

  1. Select all files to merge (Ctrl+Click)
  2. Click New > Create New Document(s) based on Selected
  3. In Import window, select all pages from one file
  4. Drag selected pages to the other document name
  5. Assign Document Type and drag all pages to it
  6. Click Store New Documents
  7. Set old documents to Inactive status

Stored Documents - Quick Reference

BYTE STORED DOCUMENTS - QUICK REFERENCE
========================================

ACCESS METHODS:
- General Menu > Stored Documents
- Filing Cabinet Icon (Main Window)
- All Screens Menu

DOCUMENT CATEGORIES:
APP (Application), DISC (Disclosures), CRED (Credit),
INC (Income), ASSET (Assets), PROP (Property),
DOCS (Documents), GOV (Government), TITLE (Title),
BOND/SUB FIN, MISC, PTF DOCS, AUDIT, CORR, WHS, POST CLSING

UPLOAD METHODS:
1. Drag & Drop - PDF files only
2. BytePro Print to Loan File - For e-signed docs
3. New > Import from Disk - Traditional upload

CRITICAL WARNING:
E-signed docs (DocuSign, DotLoop) MUST use BytePro Print.
Do NOT drag and drop - bundles will be rejected.

DOCUMENT STATUSES:
- Approved: Required for Investor/MI packages
- Reviewed: OK for internal docs
- Inactive: For duplicates/unused docs
- Not Reviewed: Default status

ANNOTATION TOOLS:
- Standard Note/Arrow: Editable by everyone
- Internal Note/Arrow: Creator only, visible to team
- Signature/Black Out/Typewriter: Burned on save

SPLIT DOCUMENTS:
1. Select doc > New > Create Based On
2. Ctrl+Click pages to separate
3. Click Split or drag to Document Type
4. Mark original as Inactive

MERGE DOCUMENTS:
1. Ctrl+Click multiple files
2. New > Create Based On
3. Drag pages to combine
4. Assign Document Type
5. Mark originals as Inactive

KEYBOARD SHORTCUTS:
Ctrl+Click = Select multiple items
Double-Click = Open edit window
Enter (in search) = Execute search
